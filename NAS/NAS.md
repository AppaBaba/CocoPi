Initial configuration involves flashing Raspberry Pi OS Lite, enabling SSH, setting hostnamelsblkPartition and format it (ext4 recommended):sudo fdisk /dev/sdasudo mkfs.ext4 /dev/sda1Mount it and make it persistent:sudo mkdir /mnt/sda1sudo nano /etc/fstab/dev/sda1 /mnt/sda1 ext4 defaults,noatime 0 1Sharing over the network can be done with Samba:sudo apt install samba samba-common-binsudo nano /etc/samba/smb.conf[shared]path=/mnt/sda1/sharedwriteable=Yescreate mask=0777directory mask=0777public=noRestart Samba and create a user:sudo systemctl restart smbdsudo adduser pi-nas-usersudo smbpasswd -a pi-nas-user
